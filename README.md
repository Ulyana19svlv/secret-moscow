# Trello places export

Мини-скрипт для выгрузки карточек из одной колонки Trello через API.

## 1. Получить Trello API key и token

1. Открой страницу API key: <https://trello.com/app-key>
2. Скопируй `Key`.
3. На этой же странице перейди по ссылке для генерации `Token` и разреши доступ.

Важно: `key` и `token` нельзя публиковать, коммитить в GitHub или вставлять в клиентский код сайта.

## 2. Узнать доску и колонку

Можно указать доску одним из способов:

- точное название доски;
- ID доски;
- short link из URL, например для `https://trello.com/b/abc123/name` это `abc123`;
- полный URL доски.

Колонку указываем по точному названию списка в Trello.

## 3. Запустить экспорт

В PowerShell:

```powershell
$env:TRELLO_KEY="your_api_key"
$env:TRELLO_TOKEN="your_api_token"
$env:TRELLO_BOARD="Название доски или https://trello.com/b/..."
$env:TRELLO_LIST="Название колонки с местами"

python .\scripts\export_trello_list.py
```

Файлы появятся в папке `data/`:

- `trello_places_raw.json` - полные данные карточек;
- `trello_places.csv` - удобная таблица для просмотра и дальнейшей обработки.

## 4. Что выгружается

Скрипт сохраняет:

- название карточки;
- описание;
- ссылку на карточку Trello;
- метки;
- все URL, найденные в описании;
- вложения;
- чеклисты;
- дату последней активности.

## 5. Сайт

Первая версия сайта находится в папке `site/`.

Для запуска локально:

```powershell
python -m http.server 5173 -d .\site
```

Затем открыть:

```text
http://localhost:5173
```

Карта сделана под Яндекс.Карты. Для интерактивной карты нужен отдельный API-ключ Яндекс.Карт JavaScript API. Trello-ключ туда вставлять не нужно.

## 6. GitHub Pages

Для публикации на GitHub Pages:

1. Создать репозиторий на GitHub.
2. Запушить этот проект в репозиторий.
3. В настройках репозитория открыть `Settings` -> `Pages`.
4. Выбрать `Deploy from a branch`, ветку `master` или `main`, папку `/root`.
5. Открыть выданную GitHub Pages ссылку.

Корневой `index.html` автоматически перенаправляет на сайт в папке `site/`.

## 7. Приемка новых мест

Для разбора входящих сообщений от Маши есть локальная страница:

```text
tools/inbox.html
```

Ее можно открыть двойным кликом в браузере. Вставь туда пачку сообщений из Instagram или Telegram, разделяя разные места пустой строкой, и нажми `Разобрать`. Страница покажет черновики и даст скачать `places_inbox.json`.

Альтернативно можно положить текст в файл:

```text
data/inbox_text.txt
```

И запустить:

```powershell
python .\scripts\import_inbox.py
```

Черновики сохранятся в:

```text
data/places_inbox.json
```

Папка `data/` не публикуется на GitHub, поэтому входящие заметки остаются локальными.
